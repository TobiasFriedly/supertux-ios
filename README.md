# SuperTux iOS Port

Open `SuperTux-iOS.xcodeproj` in Xcode.

Select the `SuperTux iOS` target, set your Team and Bundle Identifier under Signing & Capabilities, then build for an iPhone or simulator. 

The full SuperTux source is in `ios port/source/supertux/`.
